# مرحله ۲: ورود و نمایش کاربر فعلی

## هدف
مطمئن شویم روند ورود با کد پیامکی و نمایش اطلاعات کاربر در منو درست کار می‌کند.

## کارها
- [ ] در صفحه ورود، شماره موبایل خودت را وارد کن و دکمهٔ دریافت کد را بزن.
- [ ] کد پیامکی را در چهار خانه وارد کن و وارد برنامه شو.
- [ ] بعد از ورود، بررسی کن:
  - [ ] به صفحه داشبورد منتقل شده‌ای.
  - [ ] در منوی سمت راست، بخش «کاربر فعلی» دیده می‌شود.
  - [ ] شماره موبایل تو به‌طور کامل نمایش داده می‌شود (بریدگی نداشته باشد).
  - [ ] نقش کاربر (مثلاً مدیر کل یا بازاریاب) درست نوشته شده است.
