# مرحله ۱: بررسی ظاهر در موبایل

## هدف
اطمینان از این‌که صفحه‌ها در موبایل راحت دیده و استفاده می‌شوند.

## کارها
- [ ] صفحه اصلی را در موبایل باز کن و ببین همه چیز در صفحه جا می‌شود و نیاز نیست به دو طرف صفحه اسکرول کنی.
- [ ] روی دکمه‌ها در موبایل بزن و مطمئن شو به‌راحتی لمس می‌شوند و خیلی ریز نیستند.
- [ ] منوی بالای صفحه (سه خط) را باز و بسته کن و ببین:
  - [ ] منو کامل دیده می‌شود.
  - [ ] پس‌زمینه کمی تیره می‌شود.
  - [ ] با زدن روی بیرون منو، منو بسته می‌شود.
- [ ] اگر پنجرهٔ کوچک (مودال) باز می‌شود، بررسی کن:
  - [ ] وسط صفحه باز می‌شود.
  - [ ] اگر متن زیاد است می‌توانی داخل آن اسکرول کنی.
  - [ ] دکمه بستن پیدا است و کار می‌کند.
  - [ ] با زدن روی پس‌زمینه، پنجره بسته می‌شود.
